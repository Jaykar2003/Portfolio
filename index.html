Grocery Shoap

This project is a fully responsive grocery shop web page built using **HTML**, **CSS**, and **JavaScript**. The shop features an interactive product catalog where users can browse items, filter by category, and add products to a shopping cart. JavaScript handles dynamic functionalities like adding items to the cart, calculating totals, and validating forms during checkout. The design is styled using CSS for an attractive and user-friendly interface, with a responsive layout for all screen sizes. Features include a search bar, product sorting, real-time cart updates, and a checkout process with options for delivery or in-store pickup.
